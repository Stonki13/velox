=== CONTEXT: WHAT VELOX ALREADY HAS ===
C++17 rigid-body engine. Shapes: sphere, box, capsule, cylinder, cone, convex hull (QuickHull, exact mass), compound, static plane/triangle-mesh (BVH)/heightfield. Pipeline: speculative contacts + solver substeps (TGS-style live gaps) + split-impulse position correction + conservative-advancement CCD safety net. Joints: ball, distance, spring, hinge, cone-twist, fixed, prismatic, 6DoF, with motors/limits/breaking. Sleeping/islands, contact events/sensors/filtering, contact modification callback, world snapshots/rollback, origin shifting, debug-line output, per-step stats. Queries: raycast (closest + all hits), overlaps, shape casts, closest-points, all filtered and accelerated by an incremental dynamic AABB tree. Backends: deterministic multithreaded CPU and CUDA (graph-colored solver, CUDA Graphs). CI: Windows+Linux, MSVC/GCC/Clang, ASan/UBSan, 48-test stress suite, randomized fuzzing, soak tests. Public API lives in include/velox/world.h (class World with BodyId/JointId stable handles).
